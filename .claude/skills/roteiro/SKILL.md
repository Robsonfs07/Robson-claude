# /roteiro — Roteiro de Reel (Instagram / TikTok)

## Dependências

- **Tom de voz:** `_contexto/preferencias.md`
- **Contexto do negócio:** `_contexto/empresa.md`
- **Foco atual:** `_contexto/estrategia.md`

---

## Público-alvo

Comerciantes locais — donos de pequenos negócios, lojas físicas, prestadores de serviço. Linguagem simples, exemplos concretos do dia a dia do comércio local.

---

## Workflow

### Passo 1 — Entender o input

O usuário pode passar:
- Um tema ("faz um roteiro sobre como não perder cliente")
- Um arquivo de estudo de `estudos/`
- Um carrossel já criado pra adaptar pra vídeo
- Um link ou texto

Se vier de um estudo, usar os "Ângulos de conteúdo" como ponto de partida.

### Passo 2 — Definir duração e formato

Perguntar se ainda não estiver claro:
> "Pra quanto tempo? Reel curto (30-45s) ou médio (60-90s)?"

- **30-45s:** gancho + 1 ideia central + CTA
- **60-90s:** gancho + problema + solução com 2-3 pontos + CTA

### Passo 3 — Escrever o roteiro

Formato de entrega — linha a linha com tempo aproximado:

```
[0-3s] HOOK VISUAL + FRASE DE ABERTURA
→ O que aparece na tela / o que você faz
→ Frase falada (deve parar o scroll — provocação, dado, pergunta)

[4-15s] PROBLEMA / PROMESSA
→ Por que isso importa pro comerciante agora
→ Falar de dor real, não abstrata

[16-40s] CONTEÚDO PRINCIPAL
→ A resposta, o insight, o passo a passo
→ 1 ponto por bloco de 8-10s
→ Linguagem concreta, exemplos do dia a dia do comércio

[41-45s] CTA
→ O que a pessoa deve fazer agora
→ Simples: "salva", "comenta X", "segue pra mais"
```

**Tom:**
- Como o Robson fala — informal, direto, sem enrolação
- Storytelling quando couber: "imagina que você tem uma loja..."
- Analogias concretas do mundo físico
- Sem fórmulas de youtuber ("oi galera", "não esquece de dar like")

### Passo 4 — Extras opcionais

Após o roteiro, entregar:
- **Legenda sugerida** pra acompanhar o reel (3-5 linhas + hashtags)
- **Texto pro sticker de enquete** (se quiser usar no stories depois)

### Passo 5 — Salvar

Salvar em `conteudo/reels/roteiro-[slug-do-tema]-[data].md`

Confirmar:
> "Roteiro pronto. Salvo em conteudo/reels/[nome].md
> Quer adaptar esse mesmo conteúdo pra carrossel também?"

---

## Regras

- Hook é o mais importante — se não parar o scroll nos primeiros 3s, o reel não funciona
- Sempre apresentar 2 opções de hook pra o usuário escolher
- Sem travessões (—) no texto
- Roteiro deve soar como o Robson fala, não como conteúdo genérico de marketing
- Se vier de um estudo, filtrar pelo que é aplicável pro comércio local — não tudo que é interessante é relevante pro público
